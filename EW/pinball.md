# Elliott Wave Pinball Guideline

## Overview

The Pinball technique maps Fibonacci retracement and extension levels to expected wave termination points within an Elliott Wave impulse structure. It provides a roadmap for where each wave is likely to complete based on Fibonacci ratios measured from the wave 0 origin.

## Key Pinball Rules & Guidelines

### Core Elliott Wave Rules (Inviolable)

- **Wave 2 never retraces more than 100% of wave 1**
- **Wave 3 is never the shortest** of the three impulse waves (1, 3, 5)
- **Wave 4 does not overlap** the price territory of wave 1

### Pinball-Specific Observations

- The **0.764 / 0.618** zone is critical — wave ii at 0.764 and wave 2 at 0.618 define the "launch pad" for the next impulse leg
- Wave **iii** (the full sub-wave 1-5 sequence) is the extended wave in this model, consistent with the guideline that one impulse wave is typically extended
- The **2.000** target for wave V represents a full measured move from the origin

## Practical Application

1. **Identify the origin** (0.000) — the start of the impulse pattern
2. **Watch wave i** reach ~1.236, then **wave ii** retrace to ~0.764
3. **Confirm impulse begins** when wave 1 reaches ~1.382 and wave 2 holds ~0.618
4. **Expect extension** at wave 3 to ~1.618 — this is the high-confidence trade
5. **Wave 4 shallow pullback** to ~1.236 sets up the final push
6. **Wave 5 / iii** completes at ~1.764
7. **Wave iv** corrects to ~1.382
8. **Wave V** targets the full 2.000 extension for pattern completion
