# Elliott Wave Pinball 2 Guideline

## Overview

The Pinball 2 technique maps Fibonacci retracement and extension levels to a simplified impulse wave structure. Unlike Pinball 1, waves i/ii and sub-waves 1/2 share the same Fibonacci levels — the structure then extends through sub-waves 3–5 to complete wave iii, followed by iv and v.

## Key Differences from Pinball 1

| Aspect | Pinball 1 | Pinball 2 |
|--------|-----------|-----------|
| Wave i target | 1.236 | 0.618 |
| Wave ii target | 0.764 | 0.000 |
| Wave iii target | 1.764 | 1.618 |
| Wave iv target | 1.382 | 1.000 |
| Wave V/v target | 2.000 | 2.000 |
| Sub-wave 3 target | 1.618 | 1.236 |
| Sub-wave 5 target | 1.764 | 1.618 |

---

## Core Elliott Wave Rules (Inviolable)

- **Wave 2 never retraces more than 100% of wave 1**
- **Wave 3 is never the shortest** of the three impulse waves (1, 3, 5)
- **Wave 4 does not overlap** the price territory of wave 1

---

## Practical Application

1. **Identify the origin** (0.000) — the start of the impulse pattern
2. **Wave i = sub-wave 1** reaches **0.618** — first move up
3. **Wave ii = sub-wave 2** retraces to **0.000** — back to origin (launch pad)
4. **Sub-wave 3** extends to **1.236** — first extension from origin
5. **Sub-wave 4** pulls back to **0.764** — shallow retracement
6. **Sub-wave 5** reaches **1.618** — completes wave iii (golden extension)
7. **Wave iv** corrects to **1.000** — returns to the 1:1 level
8. **Wave v** targets **2.000** — full measured move completion
